Examen Git y Github

Alumn@: [José Manuel]

  1. ¿Que es un sistema de control de versiones?

    [Es una herramienta que permite guardar y controlar los cambios realizados en archivos o proyectos a lo largo del tiempo, pudiendo volver a versiones anteriores si es necesario.]

  2. ¿Cual es la diferencia entre git y github?

    [Git es un sistema de control de versiones que funciona en local. GitHub es una plataforma en la nube que permite almacenar repositorios Git y trabajar en equipo.]

  3. ¿Cómo creo una rama en git?

    [git branch nombre_rama]

  4. ¿Cómo clono un repositorio?

    [git clone ]

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull]

  6. ¿Cómo puedo gener un tag?

    [git tag nombre_tag]

  7. Cómo hago una mezcla de ramas?

    [git merge nombre_rama]]

  8. ¿Cómo puedo moverme a un commit especifico?

    [git checkout id_commit]

  9. ¿Cómo puedo borrar una rama?

    [git branch -d nombre_rama]

  10. ¿Cómo puedo borrar un tag?

    [git tag -d nombre_tag]

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. ¿Para que sirve el comando git remote -v?

    [Muestra las direcciones (URL) de los repositorios remotos configurados.]

  13. ¿Cómo le digo a git cual es mi email y correo?

    [git config --global user.name "Tu Nombre" git config --global user.email "tuemail@email.com"]

  14. ¿Cuales son los pasos para conectar git con github?

    [Crear repositorio en GitHub: Inicializar Git:git init Añadir archivos:git add . Hacer commit:git commit -m "primer commit" conectar repositorio remoto:git remote add origin URL Subirlo:git push -u origin main]

  15. ¿Cómo puedo crear un alias en git?

    [git config --global alias.co checkout]
